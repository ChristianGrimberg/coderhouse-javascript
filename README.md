# Coderhouse Javascript

Este repositorio contiene los ejercicios y ejemplos del curso de Javascript de Coderhouse.

## Estructura del Proyecto

El proyecto está organizado por clases:

- **[Clase_01](Clase_01/)**: Introducción a JavaScript, consola, variables, tipos de datos (constantes), operadores básicos y funciones de interacción (`alert()`, `prompt()`).

## Cómo ejecutar

Para ver los ejemplos de cada clase:

1. Abre el archivo [`index.html`](./Clase_01/) de la carpeta correspondiente en tu navegador.
2. Abre la consola de desarrollador (F12 o Clic derecho -> Inspeccionar -> Consola) para ver los resultados de los scripts.
