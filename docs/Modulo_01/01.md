# 1. Configuración rápida y consola del navegador

## Herramientas esenciales para comenzar con JavaScript

### Navegador y consola de desarrollador

Todos los navegadores modernos (como Chrome, Firefox, Edge y Safari) incluyen herramientas para desarrolladores, conocidas como **DevTools**. Dentro de estas herramientas, la **pestaña Console** es donde puedes escribir y ejecutar código JavaScript directamente.

#### ¿Cómo abrir la consola?

* **Chrome y Edge**: Presiona `Ctrl + Shift + J` (Windows/Linux) o `Cmd + Option + J` (Mac).
* **Firefox**: Presiona `Ctrl + Shift + K` (Windows/Linux) o `Cmd + Option + K` (Mac).
* **Safari**: Activa primero el menú de desarrollo en preferencias, luego `Cmd + Option + C`.

> **Tip**: La consola no solo muestra errores, sino que también permite probar código al instante, lo que es ideal para aprender y depurar.

### Archivo HTML simple para ejecutar JavaScript

Para ejecutar scripts más largos o guardarlos, es común crear un archivo propio JavaScript dedicado que enrute en el HTML.

```html
<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  //Aquí va enrutado tu archivo JavaScript
  <title>Mi primer script</title>
</head>
<body>
  <h1>Hola Mundo!</h1>
</body>
</html>
```

Guarda este archivo con extensión .html y ábrelo en tu navegador. Luego, abre la consola para ver el mensaje.

### Editores de código recomendados

Para editar tus archivos HTML y JavaScript, es importante usar un editor que facilite la escritura y organización del código. Algunos editores populares y gratuitos son:

* **Visual Studio Code (VS Code)**: Ligero, con muchas extensiones para desarrollo web, el más recomendado.
* **Sublime Text**: Rápido y sencillo.
* **Atom**: Personalizable y amigable para principiantes.

Estos editores resaltan la sintaxis, ofrecen autocompletado y facilitan la navegación en proyectos.
